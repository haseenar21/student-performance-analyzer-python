# student-performance-analyzer-python
A python based mini project that analyzes student performance using **Pandas** and **Matplotlib**

## Features
- Reads student data from a CSV file
- Calculates Pass/File results
- Computes pass percentage
- Identifies top performers
- Highlights students needing attention
- Visualizes marks using bar charts
- Saves final results to a new CSV file

## Technologies used
- Python
- Pandas
- Matplotlib
